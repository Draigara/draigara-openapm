# Support

## Questions and usage

Use GitHub Discussions when enabled for architecture and usage questions. Use issues for reproducible defects and accepted feature proposals.

Before opening an issue:

- run the documented diagnostics;
- search existing issues;
- include product versions and platform;
- remove credentials, proprietary source, and private marketplace content;
- provide a minimal synthetic reproduction.

## Security

Use the private process in `SECURITY.md`. Do not post suspected vulnerabilities publicly.

## Commercial support

No commercial SLA is implied by this open repository. Draigara Cloud plans may later include commercial support under separate terms.

## Scope

Maintainers may redirect requests that belong to Microsoft APM, an AI harness, an upstream package, a Git provider, a work-management provider, or a company-private marketplace.
