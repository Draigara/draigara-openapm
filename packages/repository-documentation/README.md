# Repository Documentation

An APM skill for creating and auditing clear, user-first repository READMEs and
contribution guides.

## Install

After registering Draigara OpenAPM Community, install the skill for the targets
you use:

```console
apm install repository-documentation@draigara-openapm --target claude,codex,copilot
```

Use it when asking an agent to create, rewrite, or review `README.md`,
`CONTRIBUTING.md`, installation instructions, or repository onboarding.

The package contains instructions only. It has no scripts, hooks, MCP server,
network access, or credentials.
